# Order-Processing-System

Order Processing System is a full-stack e-commerce-style application with a Next.js frontend, a Python- FastAPI backend, and PostgreSQL persistence.
It supports customer registration, login, product browsing, cart management, checkout, and post-purchase order tracking.

## What It Does

- Customer account registration and login
- Product catalog browsing
- Cart and checkout flow
- Order creation and payment submission
- Automatic order status progression
- Order cancellation and return handling

## Stack

- Frontend: Next.js and TypeScript
- Backend: FastAPI and Python
- Database: PostgreSQL
- Migrations: Alembic
- Local orchestration: Docker Compose

## Quick Start

- Start the local stack with Docker Compose.
- Apply backend migrations before using the application.
- Open the frontend in your browser and begin with registration or login.
